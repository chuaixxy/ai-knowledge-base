# skill: tech-summary · 需求


## 要做什么
1. 对已采集的技术文文章或者开源项目生成分析摘要
2. 从knowledge/raw中读文件
3. 可选步骤：文件中没有可用信息，可以从source_url等信息中获取
4. 生成中文技术摘要，有字数限制，好的示例，和不好的。
5. 评分，按照AGENTS.md评分标准打分
6. 提取标签，先提供标签词库，优先使用，保持标签一致性。
7. 输出格式
8. 批量处理：当一个文件中包含多个条目时：
- 逐条处理，每条独立评分
- 不要因为前面的条目分数高就压低后面的（绝对评分，非相对排名）
- 处理完所有条目后再统一输出
